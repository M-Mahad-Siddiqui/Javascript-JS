DOM Manipulation

Attributes

  1. getAttribute(attr) : to get the attribute value
  2. setAttribute(attr, value) : to set the attribute value

For Style

  1. node.style : to set the style

Insert Elements

  1. node.append( el ) : adds at the end of nodes (inside)
  2. node.prepend( el ) : adds at the start of nodes (inside)
  3. node.before( el ) : adds before the node (outside)
  4. node.after( el ) : adds after the node (outside)

Delete Elements

  1. node.remove() : removes the node
