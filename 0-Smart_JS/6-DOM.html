Window Object

The window object represents an open window in a browser. It is the browser's global object (not JavaScript) and is automatically created when a new window is opened by the browser. It is a global object with lots of properties and methods. `alert`, `console`, `clearInterval`, and etc., are parts of the window object.

DOM (Document Object Model)

  • When a web page is loaded, the browser creates a document object of the page.
  • DOM is a programming interface for HTML and XML documents.
  • DOM creates a tree of objects of HTML elements.
  • All the HTML elements are represented by nodes in the tree.
  • All the HTML code is present in the document object of the page, which is part of the window object.
