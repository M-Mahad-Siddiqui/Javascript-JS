Events in JS

The change in the state of an object in know as event. Events are fired to notify code of "interesting changes" that may affect code excution.
  • Mouse Events : (click, double click etc.)
  • Keyboard Events : (keypress,keyUp keydown etc.)
  • Form Events : (Submit etc)
  • Print events & many more Events are available in JS

Event Object :

Its is a special object that has details about the event. All the handlers have access to the event object's properties and methods.

Types of Event to be apply on element

  1. inline Event : syntax "onclick = 'myFunction' "
  2. External Event : syntax node.event = "myFunction"
  3. Using EventListener() :syntax node.addEventListener("click", myCallbackFunction)
    • addEventListener('click', 'myCallBackFunction')
    • removeEventListener('click', 'myCallBackFunction')
    • **Note** the callBack reference should be same to remove (used variable to store function instead of directly create in parameter)
